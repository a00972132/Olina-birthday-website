# This directory contains photos for Olina's birthday celebration website. 

# To add or manage photos, follow these instructions:

1. Place your image files in this directory.
2. Ensure that the image files are in a web-friendly format (e.g., .jpg, .png).
3. Update the `data/photos.json` file to include the new images, specifying the "src" for the image source and a "caption" if desired.
4. Use descriptive captions to enhance the viewing experience.

# Example of a photo object in photos.json:
# {
#   "src": "photos/image1.jpg",
#   "caption": "Olina's first birthday"
# }
